COVID-19 Data Exploration and Vaccination Analysis Using SQL

Summary:
This project analyzes COVID-19 case, death, and vaccination data using SQL to uncover trends across countries, continents, and time. The analysis focuses on infection rates, death counts, global case patterns, and vaccination progress. SQL techniques such as joins, aggregate functions, window functions, CTEs, and temporary tables were used to transform raw data into meaningful insights.

Business Problem:
Large public health datasets can be difficult to interpret without structured analysis, especially when comparing case counts, death rates, and vaccination progress across locations. The goal of this project was to use SQL to organize and analyze COVID-19 data in order to identify countries with the highest infection and death rates, evaluate global trends over time, and compare vaccination progress with infection rates. This type of analysis supports clearer reporting and better understanding of pandemic patterns.

Methodology:
1. Explored COVID-19 deaths and vaccination tables to identify key fields for analysis, including cases, deaths, population, and vaccinations.
2. Calculated case and death trends by location and date to examine lethality rates and the spread of COVID-19 over time.
3. Compared countries and continents using aggregate functions to identify the highest infection rates and death counts.
4. Analyzed global daily totals by summing new cases and deaths to measure worldwide trends and death percentages.
5. Joined deaths and vaccination tables to calculate rolling vaccination counts and compare vaccination progress against infection trends.
6. Used CTEs and temporary tables to organize calculations and create reusable views of vaccination and infection metrics.

Skills:
1. SQL: Joins, CTEs, temporary tables, aggregate functions, window functions, CAST, CONVERT
2. Data Analysis: Trend analysis, rolling totals, infection rate analysis, death rate analysis, vaccination tracking
3. Data Handling: Data transformation, table joins, percentage calculations, comparative analysis across countries and continents
